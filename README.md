# clojurine

A simple port of my anigramer/margarine program which helps solve countdown conundrums.

## Installation

```
git clone URL
```

### Dependencies

Java, unfortunately.

## Usage

```
java -jar clojurine-0.1.0-standalone.jar wordlist word
```

## Examples

```
java -jar clojurine-0.1.0-standalone.jar words anigrammer
```
